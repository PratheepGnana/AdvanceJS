## JavaScript Built-in Constructors

In JavaScript, **constructors** are special functions used to create and initialize objects. Built-in constructors are provided by JavaScript itself.

---

### What is a Constructor?

A constructor is called with the `new` keyword, which:
1. Creates a new empty object
2. Sets `this` to point to that object
3. Returns the object automatically

---

### Common Built-in Constructors

**Object()**
```js
const obj = new Object(); // same as {}
obj.name = "Alice";
```

**Array()**
```js
const arr = new Array(3);     // [empty × 3]
const arr2 = new Array(1,2,3); // [1, 2, 3]
```

**String()**
```js
const str = new String("hello"); // String object (not primitive)
typeof str; // "object"
```

**Number()**
```js
const num = new Number(42); // Number object
typeof num; // "object"
```

**Boolean()**
```js
const bool = new Boolean(true); // Boolean object
typeof bool; // "object"
```

**Function()**
```js
const add = new Function("a", "b", "return a + b");
add(2, 3); // 5
```

**Date()**
```js
const now = new Date();
const specific = new Date("2024-01-01");
```

**RegExp()**
```js
const regex = new RegExp("\\d+", "g"); // same as /\d+/g
```

**Error()**
```js
const err = new Error("Something went wrong");
err.message; // "Something went wrong"
```

**Map(), Set(), WeakMap(), WeakSet()**
```js
const map = new Map();
const set = new Set([1, 2, 3]);
```

**Promise()**
```js
const p = new Promise((resolve, reject) => {
  resolve("done");
});
```

---

### Custom Constructor (how it works internally)

```js
function Person(name, age) {
  this.name = name;
  this.age = age;
}

const alice = new Person("Alice", 30);
// alice = { name: "Alice", age: 30 }
```

---

### ⚠️ Important Note

Avoid using `new String()`, `new Number()`, `new Boolean()` — they create **wrapper objects**, not primitives, which can cause unexpected behavior:

```js
const a = new Boolean(false);
if (a) console.log("Truthy!"); // runs! because objects are always truthy
```

Instead, use literals:
```js
const str = "hello";   // ✅
const num = 42;        // ✅
const bool = false;    // ✅
```
